about

What is a Decentralized Web Node?

A Decentralized Web Node (DWN) is a secure storage and a communication hub that you can use to manage your personal, public, and application data. DWNs enable completely new application architectures, in which your data is yours by design.

DWNs conform to DIF's DWN specification.

What is the Community Node?

Powered by Google Cloud and TBD, and operated by the Decentralized Identity Foundation, the Decentralized Web Node "Community" version makes it simple to get started with next-generation web infrastructure. It's designed to enable easy integration of DWNs for non-production use, encouraging exploration and development.
